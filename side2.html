Binære Tal

Moderne elektroniske datamaskiner brukert den enklaste og mest pålitliege måten å lagre informasjon på. Ein brukar talene 0 til 1 for å representara straum av og straum på. Des to fleire 0 til 1 ein bruker desto meir informasjon kan ein gi

Bits

Eit eksempel er to lampar som har av og på tilstand, altså to bit. To bit gir fire ulike beskjedar eller val.

Når me sett tala 0 og ein saman til 11, 00, 10, 01 kallar vi det binære tal. Om me aukar anntall lampar til tre, har me åtte val i eit tre bits system. Binære tal blir brukt for tilstandane av og på i ei datamasking. To bit gir fire tal og tre bit git åtte val

For å finne ut kor mange val ein bit har, tar du talet 2 og ophøger det med talet på bit. Har ein åtte bit er det det samme som ein byte. Den normale minste plassen ein kan sette av til bruk i minnet i dei fleste datamaskinene. Ein byte git 256 ulike val, det gjeld ofte fargar, lydnivå eller eit teiknsett

Lagring

Lagringsplassen blir oppgitt i byte emd ein prefliks som viser kor mang tusen eller millionar byte det er snakk om. Nokre vanlige prefleks er kilo, mega og giga Ein kilobyte er omrent 1000 byte. Me oppgit lagrin i byte og farten på nettverket i kor mange bit det er i sekundet
